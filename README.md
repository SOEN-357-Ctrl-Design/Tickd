# Tickd

A checklist app built with Expo (React Native and Firebase).

From repo root:

```bash
cd Tickd
npm install
npm start
```

From `Tickd/`, you can also start straight into a platform: `npm run ios` (iOS Simulator), `npm run android` (Android emulator), or `npm run web` (browser).

For those commands you need the native tooling: **Xcode** (macOS only) for the iOS Simulator, and **Android Studio** with an emulator (AVD) set up for Android.
